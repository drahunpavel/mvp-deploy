# React MVP Project

A minimal viable product (MVP) built with React and powered by Vite for fast development and bundling.

##  Tech Stack
- `React`
- `Vite`
- `Yarn`
- `Node.js 24.2`

##  Setup & Run

### 1. Install dependencies

```bash
yarn install
```

### 2. Start the development server

```bash
yarn dev
```
### 3. Build for production:

```bash
yarn build
```
